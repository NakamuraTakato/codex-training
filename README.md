# Modern Blog Platform

このリポジトリは、Django 5 と SQLite を用いて構築したモダンなブログプラットフォームです。ログインしたユーザーのみが投稿の作成・編集・削除を行えます。カテゴリ、タグ、検索、画像アップロードに対応しています。

## 主な機能
- アカウント登録と認証（パスワードリセット対応）
- 投稿の作成・更新・削除（ログイン必須）
- カテゴリとタグによる分類、および検索フィルタ
- 画像アップロード付きの投稿フォーム
- ブートストラップ 5 によるレスポンシブな UI

## セットアップ
1. 依存関係をインストールします。
   ```bash
   pip install -r requirements.txt
   ```
2. データベースマイグレーションを実行します。
   ```bash
   python manage.py migrate
   ```
3. 管理ユーザーを作成する場合は以下を実行します。
   ```bash
   python manage.py createsuperuser
   ```
4. 開発サーバーを起動します。
   ```bash
   python manage.py runserver
   ```
5. ブラウザで `http://127.0.0.1:8000/` にアクセスするとサイトを確認できます。

## 環境変数
- `DJANGO_SECRET_KEY`: 本番環境では安全な値を設定してください。
- `DJANGO_DEBUG`: デバッグモードを有効にする場合は `1` を指定します。
- `DJANGO_ALLOWED_HOSTS`: カンマ区切りで許可するホスト名を指定します。

## 静的ファイルとメディア
`static/` ディレクトリに配置したファイルは Django の `collectstatic` コマンドで収集され、`staticfiles/` に配置されます。画像アップロードは `media/` ディレクトリに保存されます。

## テスト
現時点では最小限のセットアップのみを提供しています。必要に応じて `blog/tests.py` にテストを追加してください。
